# Introduction

## 形容詞とは

!!! Example
    1. I like to watch **old** movies. (私は古い映画をみるのが好きだった)
    2. The lady was **charming**. (その女性は魅力的だった)
    3. I found the lecture **boring**. (私はその講義を退屈だと思った)
    4. [x] boring: 退屈な、うんざりさせる

!!! Note
    ==形容詞とは名詞の性質や状態を表す語で、名詞の前や、第2文型・第5文型の補語の一に置かれる。==  
    1の形容詞のoldは、名詞のmoviesの前に置かれてmoviesを修飾している。  
    このように ==名詞を直接修飾する形容詞の用法を限定用法と呼ぶ。==  
    2は第2文型 S+V+Cで、形容詞のcharmingは補語の位置にきて、主語のThe ladyの性質を表ている。  
    3は第5文型 S+V+O+C で、形容詞のboringは補語として用いられ、目的語のthe lectureの性質を説明している。  
    このように ==補語の一にくる形容詞の用法を叙述用法と呼ぶ。==

## 形容詞の種類

!!! Example
    1. **Some** students found the lecture interesting. (何人かの学生はその講義を面白いと思った)

!!! Note
    ==多い 少ない など、名詞の数を表す形容詞を数量形容詞と呼ぶ。==  
    例文のsomeがその例で、someは いくらかの を意味する。  
    数量形容詞の多くは、形容詞のほかにも代名詞としての働きもする。

## 形容詞の限定用法

!!! Example
    1. Paul Auster is a **famous** author. (ポール・オースターは有名な作家だ)
    2. Barcelona is a city **famous for its architecture and art**. (バルセロナは建築と芸術で有名な都市だ)
    3. [x] Paul Auster: ポール・オースター
    4. [x] author: 作家
    5. [x] Barcelona: バルセロナ
    6. [x] architecture: 建築

!!! Note
    形容詞が名詞を修飾する場合、 ==形容詞1語の場合は名詞の前に、形容詞がほか語句を伴う場合は名詞の後に置く。==  
    1と2ともfamous 有名な という形容詞を使っているが、1ではfamous1語だけで使われているので名詞authorの直前に置かれている。  
    たいして2では、famousは他の語句 for its architecture and art を伴っているので、名詞cityの後ろに置かれていることを確認しよう。

## 形容詞の叙述用法

!!! Example
    1. Kenji was still **asleep** at noon when I called him. (正午に電話をかけた時、検事はまだ眠っていた)
    2. I found Kate **alone** in the classroom. (私はけいとが教室に一人でいるところを見かけた)
    3. [x] asleep: 眠って

!!! Note
    ==形容詞の中には、叙述用法でしか使われないものがある。==  
    1のasleep 眠って 2のalone ただ一人の がその例。

## 限定用法と叙述用法で意味の異なる形容詞

!!! Example
    1. My wife and I married in our **late** 30's. (妻と私は30際代の後半に結婚した)
    2. Matt is always **late** for work. (マットはいつも仕事に遅刻する)

!!! Note
    ==形容詞の中には、限定用法と叙述用法で意味が異なるものがある。==  
    lateがその例で、限定用法の場合1は 後期の；最近の を、叙述用法の場合2は 遅れた、遅刻した を表す。

## 物・事を主語にする形容詞と人を主語にする形容詞

!!! Example
    1. The game was **exciting**. (その試合はエキサイティングだった)
    2. We were **excited** about the game. (私たちはその試合に興奮した)

!!! Note
    ==分詞から派生した、感情を表す形容詞に注意。==  
    ==現在分詞から派生した形容詞 ingで終わるもの は物・ことを主語に、過去分詞から派生した形容詞 主に-edでおわるもの は人を主語にすることが多い。==  
    exciting 興奮させる→エキサイティングな excited 興奮させられた→興奮した のように覚える。
