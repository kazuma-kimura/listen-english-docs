# Introduction

## 助動詞とは

!!! Example
    1. Kate **can** speak Japanese. (けいとは日本語を話すことができる)
    2. Robert **might** arrive soon. (ロバートはそろそろ到着するかもしれない)
    3. You **must** study herder. (あなたはもっと一生懸命勉強しなければならない)

!!! Note
    ==助動詞は 助動詞＋do 動詞の原型 の形で用いられ、可能・水量・義務などの意味は動詞に添える。==  
    1ではcan speakで 話すことができる という可能を、2ではmight arriveで 到着するかもしれない という水量を、3ではmust studyで 勉強しなければならない という義務を表ている

## 助動詞の種類

!!! Example
    1. You **ought to** apply for the job. (あなたはその仕事に応募したほうがいい)
    2. I **used to** travel a lot. (私は以前は旅行をたくさんしたものだった)
    3. Denis **could** read and write at the age of three (デニスは3歳の時には読み書きができた)

!!! Note
    ==助動詞には、1現在形だけのもの、2過去形だけのもの、3現在形と過去形の両方があるものがある。==  
    ただし、時制の一致で過去形に変える場合を覗き、過去形の助動詞でも現在の意味を表すことが多いので注意。

## 助動詞を使った肯定文の形

!!! Example
    1. You **may** go now. (もう行ってもよろしい)

!!! Note
    ==助動詞を使った肯定文は 助動詞＋do 動詞の原型 の形になる。==  
    時制が現在で、主語が3認証単数 he/she/it でも動詞に-sをつけないように注意。

## 助動詞を使った否定文の形

!!! Example
    1. I **cannot** stand his behavior!. (彼の態度には我慢できない)
    2. You **ought not to** tell lies. (うそをつくべきではない)
    3. I **didn't use to** like coffee. (私は以前はコーヒーが嫌いだった)
    4. [x] stand: 〜を我慢する

!!! Note
    ==1語の助動詞は、助動詞の後にnotをつける 1。==  
    ==ought toの否定形はought not to 2。==  
    ==used toの否定形は、didn't use toが普通 3== used not to、 usedn't toを使うこともある。

## 助動詞を使った疑問群の形

!!! Example
    1. **May** I ask you a question?. (1つ質問してもいいですか)
    2. **Ought** I **to** apologize to her? (私は彼女に謝るべきでしょうか)
    3. **Did** you **use to** smoke? (以前はよくたばこを吸いましたか)
    4. [x] May I 〜 ?:〜してもよろしいですか？
    5. [x] apologize: 謝る

!!! Note
    ==1後の助動詞は、文頭に助動詞をおく1。==  
    ==ought toの場合はOught S toの語順になる 2。==  
    ==used toの疑問文はDid S use toの語順になる 3== Used S toを使うこともある。

## 助動詞のneedとdareの用法

!!! Example
    1. You **need** not worry about it. (そのことについては心配する必要はありません)
    2. How **dare** you say such a thing to me? (よくもそんなことを私に言えますね)
    3. [x] How dare〜?:よくも[ずうずうしくも] 〜できるね

!!! Note
    ==否定文と疑問文でのみ用いられるneed 〜する必要がある とdare あえて〜する。==  
    いずれも、否定文で用いる場合は、動詞としてneed to do 〜する必要がある dare to do あえて〜する のように、目的語にto不定詞を取る。
