# Introduction

## 名詞とは 冠詞とは

!!! Example
    1. Richard has two **cars** and a **motorcycle**. (リチャードは2大の自動車と1大のオートバイを持っている)
    2. Angela prefers **wine** to **beer**. (アンジェラはビールよりワインのほうが好きだ)
    3. [x] prefer A to B: BよりAが好きである

!!! Note
    ==名詞とは人や物・車の名前を示す語である。==  
    また、 ==冠詞のa anは、名詞が数えられるか数えられないか、theは、その名詞が初めて話題に登場する語がすでに話題に出た語かなどによってつけられる。==  
    例文1では、carは数えられる名詞なので2台の場合は複数形のcarsになる。  
    また、motorcycleも数えられる名詞なので1台の場合は関しのaが前についている。  
    2のwineとbeerは数えられない名詞なので冠詞のaがついていないことを確認しよう。

## 名詞の種類1：加算名詞

!!! Example
    1. Arthur is moving to a new **apartment** in a **week**. There are five **families** in the apartment. (アーサーは1習慣後に新しいアパートに引っ越す予定だ。そのアパートには5つの家族が住んでいる)
    2. [x] move to A: Aに引っ越す

!!! Note
    ==一定の形や区切りがあり、1つの物・単位として数えられる名詞を普通名詞、いくつかの同種の人・物の集合体を表す名詞を酒豪名詞と呼ぶ。==  
    例文のapartment、weekが普通名詞、familyが集合名詞であり、 ==普通名詞と集合名詞は加算名詞 数えられる名詞 である。==  
    可算名詞には単数圭人複数形があり、複数系は普通、名詞に-sか-esをつけて作る。

## 名詞の種類2：不可算名詞

!!! Example
    1. I prefer classical **music** to pop **music**. (私はポップ音楽よりクラシック音楽のほうが好きだ)
    2. **Rain** is drops of **water**. (雨は水のしずくだ)
    3. **Dr.Smith** graduated from **Harvard University** in 1980. (スミス博士は1980年にハーバード大学を卒業した)
    4. [x] graduate from A: Aを卒業する

!!! Note
    ==一定の形を持たず、抽象概念を表す名詞を抽象名詞と呼ぶ。==  
    1のmusic 音楽 は形がないので抽象名詞である。  
    ==形はあるものの、区切ることができないものを表す名詞を物質名詞と呼ぶ。==  
    2のrainやwaterなどの液体、airやsmokeなどの気体、ironやgoldなどの個体などがその例。  
    また、 ==特定の人・物の名前を表す名詞を固有名詞と呼ぶ。==  
    3のDr.SmithやHarvard Universityなどがその例。  
    固有名詞は大文字で書き始め、普通はa an theなどの冠詞はつかない。  
    ==抽象名詞・物質名詞・固有名詞は不可算名詞 数えられない名詞 である。==

## 名詞の所有格

!!! Example
    1. This is **Richard's motorcycle**. (これはリチャードのオートバイだ)
    2. **The roof of my home** was blown away by the strong winds. (強風で私の家の屋根が吹き飛ばされた)
    3. [x] blow away A: Aを吹き飛ばす

!!! Note
    ==A 人間・生物 のB は、名詞AにアポストロフィとsをつけてA's Bで表す。==  
    1のRichard's motorcycleがその例。  
    ==A 無生物 B は、B or A で表す== のが原則。  
    2のthe roof of my houseでは、my houseは物 無生物 なので B of Aの形を取っている。  
    ただし、地名・天体・時間・距離・金額・重量などを表す名詞の場合は、Japan's future 日本の未来 のようにA's Bの形を取ることができる。

## 不定冠詞aとanの用法

!!! Example
    1. I want a new PC. (私は新しいパソコンが欲しい)
    2. Once upon a time there was a girl named Cinderella. (昔々ある所に、シンデレラという名前の女の子がいました)
    3. [x] PC:パソコン
    4. [x] once upon a time: 物語の出だしで 昔々、ある時に

!!! Note
    ==不定冠詞のaは可算名詞の単数形につき、不特定の1人の人・1つの物、初めての話題に上る人・物を指す場合に使う。==  
    母音の発音で始まる名詞にはanがつく。  
    1の場合は、PCは不特定の物を指しているのでaがついている。  
    2では、シンデレラという名前の女の子は初めて物語の中で登場しているので、a girlのようにaがつく。

## 定冠詞theの用法

!!! Example
    1. Arthur moved to a new apartment last week. There are five families in **the** apartment. (アーサーは新しいアパートに先週引っ越した。そのアパートには5つの家族が住んでいる)
    2. Lock **the** door when you go out. (階出する時はドアに鍵をかけなさい)

!!! Note
    ==定冠詞のtheは可算名詞・不可算名詞のどちらにもつき、既に話題に上がった人・物、状況から分かる人・物を示す場合に使う。==  
    1では、初めて話題に上がった時は不定冠詞のaがついていたapartmentが、次に使われたときにはthe apartmentとなっていることを確認しよう。  
    2では、話してと聞き手の両者が、ドアが玄関のドアだとわかっているので、the doorのように定冠詞theがついている。
