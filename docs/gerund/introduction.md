# Introduction

## 動名詞とは

!!! Example
    1. **Exercising** is good for your health. (運動することは健康にいい)
    2. Kenji admitted **stealing** the money. (けんじはそのお金を盗んだことを認めた)
    3. [x] admit: 〜を認める、自白する

!!! Note
    ==動名詞とは、動詞の性質を備えた名詞である。==  
    動名詞はdoing 動詞＋ing の形で 〜すること、〜したこと を表す。  
    名前の通り、動名詞は名詞の性質を持ち、文中で名詞と同じような働きをする。  
    1ではExercising 運動すること は文の主語になっている。  
    2では、stealing 盗んだこと は動詞のadmitted 〜を認めた の目的語になっている。

## 動名詞の性質

!!! Example
    1. Kazuo denied **stealing** the money. (かすおはそのお金を盗んでないと言った)
    2. **Seeing** is **believing**. (見ることは信じることである)
    3. [x] deny: 〜を否定する

!!! Note
    ==動名詞は 過去起きたこと、一般的・具体的・習慣的なこと を表す。==  
    1は 過去に起きたこと を表す動名詞で、stealing 盗んだこと はdenied 〜を否定した の目的語になって 盗んだことを否定した→盗んでないと言った を表ている。  
    2は、一般的・具体的なことを表す動名詞の例である。

## 動名詞の基礎用法

!!! Example
    1. **Driving** in rain is very different from **driving** in dry condtions. (雨の中を運転するのは、乾燥状態で運動するのとはかなり違う)
    2. My hobby is **collecting** antiques. (私の趣味は骨董品を集めることだ)
    3. She finished **writing** the report. (彼女はその報告書を書き終えた)
    4. [x] antique: 骨董品

!!! Note
    ==動名詞は、文の主語や補語、動詞・前置詞の目的語になる。==  
    1の最初のDrivingは文の主語、2番目のdrivingは前置詞fromの目的語になっている。  
    2では、collectingは文の補語の働きをしている。  
    3のwritingは動詞のfinishedの目的語となっている。

## 動名詞の形

!!! Example
    1. Lucy regrets **not having bought** the dress. (ルーシーはそのドレスを買わなかったことを後悔している)
    2. I hate **being laughed at**. (私は笑われるのが嫌いだ)
    3. [x] regret: 〜を後悔する
    4. [x] hate: 〜をひどく嫌う

!!! Note
    ==動名詞の否定形はnot doing 完了形はhaving done 受動形はbeing doneで表す。==  
    1ではnot having bought 買わなかったこと のように、動名詞の否定形と完了形が両方使われている。  
    完了形の動名詞は、文の動詞が表す時制 ここでは現在 よりも前の時を表す。  
    2が受動形の動名詞の例で、ここではbeing laughed atで 笑われること を表ている。

## 動名詞とto不定詞の共通点と相違点

!!! Example
    1. **Seeing [To see]** is **believing [to believe]**. (見ることは信じることである)
    2. My daughter likes **playing** **[to play]** the piano. (私の娘はピアノを引くのが好きだ)
    3. I'm interested in **seeing** [(×) **to see**]the movie. (私はその映画をみたいと思っている)

!!! Note
    ==文の主語や補語、動詞の目的語になっている動名詞は、to不定詞に置き換えることができる場合がある。==  
    ただし、2はto不定詞に置き換えても意味はほとんどかわらないが、1ではニュアンスが少々異なる。  
    また、 ==to不定詞は前置詞の目的語になれない== ことも覚えておこう。  
    3がその例で、前置詞inの後にはto不定詞を置くことはできない。  
    このほかにも、動名詞にはto不定詞のように形容詞的用法や副詞的用法がないなどの違いがある。

## 動名詞を用いた慣用表現

!!! Example
    1. The real estate agent **had trouble selling** the building and had to lower the price. (その不動産業者はビルを売却することができず、価格を下げなければならなかった)
    2. **There is no telling** what might come next in the present political situation. (現在の政治状況では次に何が起こるか分からない)

!!! Note
    ==動名詞を使った慣用表現も 固まり で覚える。==  
    直訳すれば、1のhave trouble in doingは 〜することにおいて困難がある 2のthere is no doingは 〜することは全くない となり、そこから転じて 〜するのが困難である 〜することはできない という意味になる。  
    どちらのdoingも動名詞だが、文法的なことはあまり気にしないで、熱語として 固まり で覚えるようにしましょう。
