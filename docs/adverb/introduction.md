# Introduction

## 副詞とは

!!! Example
    1. My daughter came home **late tonight**. (娘は今夜、遅く帰宅した)

!!! Note
    ==動詞を修飾するのが副詞の主な働きである。==  
    例文のlate 遅く 、tonight 今夜は が副詞で、いずれも動詞のcameを修飾している。  
    このように動詞を修飾する副詞は いつ、どこで、どのように、どのくらいの頻度で、どの程度 を表す。

## 副詞の用法

!!! Example
    1. Gary loves Cindy **deeply**. (ゲーリはシンディを深く愛している)
    2. This comic book is **very** interesting. (この漫画はとても面白い)
    3. Hiroko works **pretty** hard. (ひろこはとても熱心に働く)
    4. **Fortunately**, no one was injured in the incident. (幸いにも、その事故で誰も怪我をしなかった)
    5. [x] incident: 事故 通例、accidentよりも小さい事故を表す

!!! Note
    ==副詞は、動詞以外にも形容詞・ほかの副詞・文全体も修飾する。==  
    1の副詞のdeeply 深く が動詞のlovesを修飾している例。  
    ここでは どのように という 様態 を表ている。  
    2は形容詞を修飾する副詞の例で、副詞のvery とても はinteresting 面白い を修飾している。  
    3では、副詞のpretty とても は副詞のhard 熱心に を修飾している。  
    4は副詞のfortunetely 幸いにも が文全体を修飾している例である。  
    この4つの用法が副詞の基本的な働きである。

## 副詞の形の基本

!!! Example
    1. Hiroshi read the poem **slowly**. (ひろしはその詩をゆっくりと朗読した)
    2. I **often** visit my grandparents. (私は祖父母をよく訪れる)
    3. It will be sunny **today**. (今日は晴天になるでしょう)

!!! Note
    ==多くの形容詞は語尾に-lyがつくと副詞になる。==  
    1がその例で、形容詞のslow 動作などが遅い に-lyがついて、副詞のslowly ゆっくり になっている。  
    このほかにも副詞には、2のoften しばしば のように-lyがつかないものや、3のtoday 今日は のように名詞やほかの瀕死から派生したものもある。

## 動詞を修飾する副詞の種類と位置の原則

!!! Example
    1. Please read the instructions **carefull**. / Put the box **there**. (使用説明書を注意深く読んでください / その箱をそこに置きなさい)
    2. Jack is going to the movies hith his friends **tonight**. (ジャックは今夜、友達と一緒に映画を見に行く予定だ)
    3. I **almost** finished my homework. / I **always** get up at 6:40 a.m. (私は宿題をほとんどおわらせた / 私はいつも午前6時40分に起きる)

!!! Note
    ==動詞を修飾する副詞の一は比較的自由だが、一定のルールがある。==  
    様態・場所を表す副詞は、自動詞の場合は直後、他動詞の場合は目的語の直後に置くのが普通 1。  
    時を表す副詞は、文末におくことが多い 2。  
    程度・頻度を表す副詞は、一般動詞の前、be動詞・助動詞の後に置くのが原則である 3。

## 形容詞・副詞を修飾する副詞の位置

!!! Example
    1. It was **very** windy yesterday. (昨日はとても風が強かった)
    2. She solved the problem **quite** easily. (彼女はその問題をいとも簡単に解いた)

!!! Note
    ==形容詞・副詞を修飾する副詞は、修飾する後の前に置く。==  
    1では副詞のveryは形容詞のwindy 風邪の強い の前に、2では副詞のquite かなり は副詞のeasily 容易に の前にそれぞれ置かれている。

## 副詞のenoughは形容詞・副詞の後に置く

!!! Example
    1. Bill is rich **enough** to buy the car. (ビルはその車が帰るほどの金持ちだ)

!!! Note
    ==副詞のenoughは修飾する形容詞・副詞の後ろに置く。==  
    A 形容詞・副詞 enough to do 〜するのに十分なほどA という用法も合わせて覚えておきたい。
