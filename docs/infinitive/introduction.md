# Introduction

## 不定詞とは

!!! Example
    1. Yoshio decided **to study** abroad. (よしおは留学しようと決心した)
    2. Clara helped **cook** dinner. (クララは夕食を作るのを手伝った)
    3. [x] decide to do: 〜しようと決心する
    4. [x] help (to) do: 〜するのを手伝う

!!! Note
    ==不定詞は、to do 動詞の原形 の形で名詞・形容詞・副詞のような働きをする。==  
    この中で最も理解しやすいのは名詞の働きをする不定詞で 〜すること を意味する。  
    1ではto study 勉強すること が、2ではcook 料理すること が、それぞれdicide 〜を決心する、help 〜を手伝う の目的語になっている。  
    不定詞は動詞の形で名詞・形容詞・・副詞の働きをすることから準動詞と呼ばれることもある。  
    準動詞には、不定詞のほかにも、動名詞や分詞がある。

## 不定詞の種類1：原形不定詞

!!! Example
    1. The doctor made me **quit** smoking. (その医者は私にたばこを吸うのをやめさせた)
    2. I heard someone **call** my name. (誰かが私の名前を呼ぶのが聞こえた)
    3. [x] make A do: Aに〜させる
    4. [x] quit: 〜をやめる
    5. [x] hear A do: Aが〜するのが聞こえる

!!! Note
    ==toなしで使う不定詞を原形不定詞と呼ぶ。==  
    助動詞の後には動詞の原形が来るが、この動詞も文法的には原形不定詞である。  
    このほかに原形不定詞には使役動詞・知覚動詞の後で用いられる用法がある。  
    1は使役動詞のmake、2は知覚動詞のhearとともに原形不定詞が用いられている例である。

## 不定詞の種類2：to不定詞

!!! Example
    1. Dennis forgot **to lock** the door when he left. (デニスは出かける時にドアに鍵をかけるのを忘れた)
    2. Mayumi was surprised **to hear** the news. (まゆみはそのニュースを聞いて驚いた)
    3. I have a lot of work **to do**. (私にはしなければならない仕事がたくさんある)

!!! Note
    ==to付きで使う不定詞をto不定詞と呼ぶ。==  
    原形不定詞の用法が限られているのに対し、to不定詞はいろいろな品詞の後について、さまざまな働きをする。  
    1は、to lock 鍵をかけること が動詞のforgotの目的語になっている例。  
    2では、to hear 聞いて が形容詞のsurprisedを修飾している。  
    3のto do は すべき という意味で、直前の名詞のworkを修飾している。

## to不定詞の性質

!!! Example
    1. I intend **to study** medicine in college. (私は大学で医学を勉強するつもりだ)
    2. My dream is **to be** a doctor. (私の夢は医者になることだ)
    3. [x] intend to do: 〜するつもりである
    4. [x] medicine: 医学

!!! Note
    ==to不定詞は これからすること を表す。==  
    to不定詞にはさまざまな用法があるので、その性質をひとくくりにすることはできないが、特に名詞の働きをする場合には これからすること を表すことが多い。  
    1では、to study 勉強すること が動詞intendの目的語になって、将来にするつもりのことを表ている。  
    2では、to be a doctor 医者になること は文の補語になって、将来の夢の内容を説明している。

## to不定詞の用法

!!! Example
    1. I want **to visit** Peru in the future. (私は将来ベルーに行ってみたい)
    2. I need someone **to help** me. (私は私を手伝ってくれる人を必要としている)
    3. I went to the bank **to withdraw** money. (私はお金を下ろすために銀行へ行った)
    4. withdraw: 預金 を引き出す

!!! Note
    ==to不定詞には、名詞的用法・形容詞的用法・副詞的用法の3つの用法がある。==  
    1はto不定詞の名詞的用法で、to visit 訪れること がwant 〜を望む の目的語になっている want to do で 〜することを望む、〜したいと思う を表す。  
    2はto不定詞の形容詞的用法の例。  
    ここではto help 手伝うへき が直前の代名詞のsomeone 誰か を修飾し、someone to help meで 私を手伝うべき誰か 私を手伝ってくれる人 を表ている。  
    to help が後ろからsomeoneを修飾している点に注意使用。  
    3ではto withdraw 引き出すために は動詞のwent 行った を修飾しているので、副詞的用法である。  
    to不定詞の名詞的用法は 〜すること、形容詞的用法は 〜すべき、〜するための、副詞的用法は 〜するために と大雑把には訳せるが、特に形容詞・副詞的用法では必ずしもそうとは限らないので注意が必要である。

## to不定詞の形

!!! Example
    1. He promised **not to tell** anyone about it. (彼はそのことについては誰にも言わないと約束した)
    2. Nelson pretended **to be doing** his homework. (ネルソンは宿題をしているふりをした)
    3. Beth seems **to have lost** weight. (ペスはやせたように見える)
    4. I don't like **to be treated** like a kid. (私は子供のように扱われたくない)
    5. [x] promise to do: 〜すると約束する
    6. [x] pretend to do: 〜するフリをする
    7. [x] seem to do: 〜するように思われる
    8. [x] lose weight: やせる

!!! Note
    ==to不定詞の否定形はnot to do、進行形はto be doing、完了形はto have done、受動系はto be doneで表す。==  
    1はnot to tellで言わないこと、2はto be doingで していること、3はto have lost weightで やせた、4はto be treatedで 扱われること を表ている。  
    完了形のto不定詞は、文の動詞が表す時制よりも前の時を表すことに注意しよう。
