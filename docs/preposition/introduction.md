# Introduction

## 前置詞とは

!!! Example
    1. I leave **for** school **at** eight every day. (私は毎日8時に学校へ行く)
    2. There were many people **in** the amusement park **on** that day. (その日は遊園地にはたくさんの人がいた)
    3. [x] amusement park: 遊園地

!!! Note
    ==前置詞は主に名詞 相当語句 の前に置かれ、 前置詞＋名詞 の固まりで時・場所・理由・目的などを表す。==  
    前置詞の後に来る語 句 を前置詞の目的語と呼ぶ。  
    例文では、for schoolで 学校へ in the amusement parkで 遊園地に という場所を表ている。  
    また、at eightで 8時に on that dayで その日は という時を表ている。

## 前置詞の種類

!!! Example
    1. I was late **because of** the traffic accident. (私は交通事故の為に遅刻した)
    2. There is a bus stop **in front of** the hotel. (ホテルの前にバス停がある)
    3. [x] traffic accident: 交通事故
    4. [x] bus stop: バス停

!!! Note
    ==数語で1つの前置詞の働きをするものを郡前置詞と呼ぶ。==  
    1のbecause of Aは Aの理由で のように理由を、2のin front of Aは Aの前で[に] のように場所を表す郡前置詞である。

## 前置詞の表す意味

!!! Example
    1. She was surprised **at** the news. (彼女はそのニュースに驚いた)
    2. What do you want **for** dinner? (夕食に何を食べたいですか)
    3. I commute **by** train. (私は電車で通勤している)
    4. This spoon is made **of** silver. (このスプーンは銀でできている)
    5. [x] commute: 運動する

!!! Note
    ==前置詞＋名詞 の表す意味は、時や場所のほかにも、原因 1・目的 2・手段 3・材料 4などいろいろとある。==  
    また、基本9前置詞など、前置詞の多くは複数の意味・用法がある。  
    それぞれの前置詞が持つ基本的なイメージを押さえて、さまざまな用法をマスターしておこう。

## 前置詞を含む句の働い1：副詞句を作る

!!! Example
    1. My wife and I went **to** Ginza **for** dinner yesterday. ()
    2. Bob is good **at** mathematics. ()
    3. **To** my delight, I passed the entrance exam. ()
    4. [x] mathematics: 数学
    5. [x] to A's delight: Aが 喜んだことには、 Aにとって うれしいことには
    6. [x] entrance exam: 入学試験

!!! Note
    ==前置詞に導かれた句は副詞句を作り、動詞や形容詞を修飾する。==  
    1では、to Ginza for dinnerは動詞went 行った を修飾している。  
    2では、at mathematics 数学が は形容詞good 得意な を修飾している。  
    また、3のTo my delightのような ==慣用表現が文全体を修飾することもある。==

## 前置詞を含む句の働き2：形容詞句を作る

!!! Example
    1. This is a question **of** great importance. (これは飛香に重要な問題だ)
    2. My grandfather is **in** good health. (私の祖父は健康だ)
    3. They found the elevator **out of** order. (彼らはそのエレベーターが故障しているのに気づいた)
    4. [x] of importance: 重要な
    5. [x] out of order: 故障して

!!! Note
    ==前置詞に導かれた句は形容詞句を作り、名詞を修飾したり文の補語になる。==  
    1では、of great importance 非常に重要な は直前の名詞a question 問題 を修飾している。  
    2のin good healthは 健康な という意味で、文の補語になっている。  
    また、3のout of order 故障して のように、第5文型で目的語の補語になることもある。

## 基本9前置詞の用法

!!! Example
    1. He is **at** work now. (彼は今、仕事中だ)
    2. This is a book **on** Chinese history. (これは中国史に関する本だ)
    3. Nick is **from** New York. (ニックはニューヨーク出身だ)
    4. I want a house **with** a garage. (私は車庫つきの家が欲しい)
    5. I'm older than my wife **by** four years. (私は妻より4歳年上だ)

!!! Note
    ==基本9前置詞には様々な用法がある。==
