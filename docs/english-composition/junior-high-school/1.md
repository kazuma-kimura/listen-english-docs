# 中学1年生レベルの英作文

## This[That] is 〜 . の文

???+ Note "This is 〜 . / That is 〜 . の文"
    ??? Example "これは私の帽子です。"
        This is my hat.
    ??? Example "あれはバスです。"
        That is a bus.
    ??? Example "こちらはデイビッドです。"
        This is David.

???+ Note "否定文と疑問文"
    ??? Example "これは箱ではありません。"
        This isn't a box.
    ??? Example "これは箱ですか。"
        Is this a box?
    ??? Example "はい、そうです。 / いいえ、ちがいます。"
        Yes, It is. / No, It isn't.

???+ Note "this 名詞 / that名詞"
    ??? Example "この家はあたらしい。"
        This house is new.
    ??? Example "あの花は美しい。"
        That flower is beautiful.

???+ Note "What is 〜 ？ の文"
    ??? Example "これは何ですか？"
        What's this?
    ??? Example "あれは何ですか？"
        What's that?

## 一般動詞の文

### 肯定文

???+ Note "一般動詞の文"
    ??? Example "私は本を読みます。"
        I read a book.
    ??? Example "あなたはギターを弾きます。"
        You play the guitar.

???+ Note "動詞を修飾する副詞"
    ??? Example "私は図書館で本を読みます"
        I read a book in the library.
    ??? Example "あなたは毎日朝食を食べます。"
        You eat breakfast every day.

???+ Note "定冠詞the"
    ??? Example "私は猫を飼っています。その猫はかわいいです。"
        I have a cat. The cat is pretty.
    ??? Example "私は毎日ピアノを弾きます。"

### 否定文

???+ Note "一般動詞の文の否定文"
    ??? Example "私は辞書を持っていません。"
        I don't have a dictionary.
    ??? Example "あなたは彼の名前を知りません。"

### 疑問文

???+ Note "一般動詞の文の疑問文"
    ??? Example "あなたは大阪出身ですか。"
        Are you from Osaka?
    ??? Example "あなたは自転車を持っていますか。"
        Do you have a bike?
    ??? Example "はい、持っています。／いいえ、持っていません。"
        Yes, I do. / No, I don't.

???+ Note "選択疑問文"
    ??? Example "これは学校ですか、それとも図書館ですか。"

    ??? Example "これは図書館です。"

    ??? Example "あなたは犬が好きですか、それとも猫が好きですか。"
        Do you like a dog or a cat?
    ??? Example "私は猫が好きです。"
