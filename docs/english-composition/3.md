# 中学3年生レベルの英作文

## 文型

???+ Note "第1文型 SVM"
    ??? Example "私たちは毎朝、駅へ歩いて行きます。"
        We walk to the station every morning.
    ??? Example "彼女は両親と東京に住んでいます。"
        She lives in Tokyo with her parents.

???+ Note "第2文型 SVC"
    ??? Example "彼は良いサッカー選手です。"
        He is a good soccer player.
    ??? Example "私は疲れています。"
        I'm tired.
    ??? Example "彼女は疲れているように見える。"
        She looks tired.
    ??? Example "ナンシーは大学生のように見える。"
        Nancy looks like a college student.

???+ Note "第3文型 SVO"
    ??? Example "彼女は私をよく知っている。"
        She knows me well.
    ??? Example "ジョンは、このデスクを作った。"
        John made this desk.

???+ Note "第4文型 SV人物"
    ??? Example "私の父は、私にカメラをくれた。"
        My father gave me a camera.
    ??? Example "田中先生は、私たちに英語を教えます。"
        Mr.Tanaka teaches us English.
    ??? Example "彼は、私にメアリーが試験に合格したことを教えてくれた。"
        He told me that Mary passed the exam.

???+ Note "第5文型 SVOC"
    ??? Example "私の友達は私をアンディと呼びます。"
        My friends call me Andy.
    ??? Example "そのニュースは私たちを幸せにした。"
        The news made us happy.

    ??? Example "私は彼にその歌を歌うように頼んだ。"

    ??? Example "私は医者になりたい。"

    ??? Example "この本はあまりに難しすぎて彼には読めない。"

    ??? Example "この本はとても難しくて彼はそれを読むことができない。"

    ??? Example "その問題はとても難しくて私たちには解くことができません。"

???+ Note "...enough to 〜構文 ーが〜するには十分…だ ーは…にも〜する"
    ??? Example "彼はその家が買えるほど十分裕福です。"
        He is rich enough to buy the house.
    ??? Example "彼女は親切にも私に駅への道を教えてくれた。"
        She's kind enough to tell me the way to the station.
    ??? Example "この本は私が理解できるくらいに易しかった。"

???+ Note "so...that S (can) V構文 とても…なのでSは〜する(ことができる)"
    ??? Example "このゲームはとても簡単で子供達でも遊ぶことができる。"

    ??? Example "彼はとても親切なので彼は私たちを手伝った。"
        He was so kind that he helped us.
    ??? Example "彼はとても親切なので彼は私を駅に連れて行った。"

        Look at the girl and her dog that are crossing the street.

### 目的格の関係代名詞

???+ Note "目的格の関係代名詞whomとthat"
    ??? Example "こちらは私が駅で会った少年です。"
        This is a boy whom I met at the station.
    ??? Example "私たちがパーティーに招待したその少年はなおこです。"
        The boy whom we invited to the party is Naoko.
    ??? Example "昨日君が見かけた少女は私の妻です。"
        The woman that you saw yesterday is my wife.

???+ Note "目的格の関係代名詞whichとthat"
    ??? Example "これらはあやこがカナダでとった写真です。"
        These are the pictures which Ayako took in Canada.
    ??? Example "私の父が私に買ってくれたその本はおもしろかったです。"
        The book which my father bought for me was interesting.
    ??? Example "これはあなたが書いた本ですか。"
        Is this the book which you wrote?
    ??? Example "彼が公園で見つけた猫は病気でした。"
        The cat which he found in the park was sick.

???+ Note "関係代名詞の省略"
    ??? Example "これは父が毎日使うコンピューターです。"
        This is the computer my father uses every day.
    ??? Example "あなたがこの図書館で見た男の人は私のおじです。"
        The man you saw in this library is my uncle.
